Project 1

Hydrology Museum

Concept & Goal

This project explores how to make hydrological knowledge more engaging in museum spaces, addressing the lack of real interaction in traditional science communication. It is designed to work in combination with an interactive mobile app.

Design & Interaction

It features a hydrological identity test and sound installations simulating floods. Inspired by interactive ritual chains, it fosters community engagement and emotional connection.

Outcome & Future Work

The system shows strong potential for immersive learning, with future work focusing on cost, location adaptability, and app integration.

Project 2

Attention Economy

This project visualizes how human attention creates value in the digital age. Using a plant-based installation, it converts attention into light, symbolizing energy flow and value creation.

Project 3

UI / App Design

This UI design uses green as the main color, highlighting the theme of environmental sustainability. The creative focus explores the cyclical flow of knowledge and human capability.
